# Marketplace Copy

Reusable listing copy for skills marketplaces, GitHub descriptions, and launch posts.

## English

### One-line

Windows-first guardrails for AI agents and terminal-capable assistants.

### Short Listing

Prevent common PowerShell and CLI failures before execution with reusable command patterns for quoting, tool discovery, UTF-8 inline Python, search, traversal, and file operations.

### Medium Listing

`windows-agent-guardrails` is a Windows-first skill for AI agents and terminal-capable assistants that need safer PowerShell and CLI execution. It helps reduce common command-construction failures with reusable patterns for quoting paths, checking tool availability, handling UTF-8 inline Python, performing search and traversal, and using safer native file-operation shapes.

### Medium Listing With Representative Agents

Designed for terminal-capable workflows built around Codex, OpenClaw, Claude Code, OpenCode, Cline, Goose, and similar agent systems. It can also support tool-enabled ChatGPT or Gemini workflows on Windows when they actually reach terminal execution.

### Tagline Options

- Safer PowerShell and Windows CLI execution for AI agents.
- Reusable Windows command guardrails for terminal-capable assistants.
- Fewer quoting, path, and tool-discovery failures in Windows agent workflows.
